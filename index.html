Project Proposal

The project addresses a practical and growing risk: important digital and legal information is often scattered across email accounts, devices, paper folders, and cloud drives, making retrieval difficult during emergencies or after death. This MVP provides a controlled workflow where an owner creates a vault profile, uploads key documents, assigns a trusted executor, and tracks readiness status through a web interface. An administrator reviews submissions, validates required fields, and manages lifecycle states for each record.

The platform will be deployed on an IaaS virtual machine and exposed through a controlled domain with HTTPS enabled. The architecture includes a web application layer, encrypted document storage, a relational database for metadata, and an audit logging component that records uploads, status changes, permission updates, and access events. A role-based access model separates owner, executor, and administrator permissions. Phase 1 will include a release simulation mode where executor access is read-only and only available after admin approval, supporting safe demonstration without implementing legal trigger automation.

A scripting component will automate daily backup, log rotation, and integrity checks. The script will generate timestamped reports for backup success, checksum validation, storage usage, and recent security events, providing verifiable evidence of useful automation beyond basic lab tasks.

Digital Legacy Vault Phase 1 is a strong fit because it is unique, realistic, and aligned with long-term portfolio development. It demonstrates Linux server setup, web hosting, DNS configuration, SSL/TLS implementation, secure file handling, role-based control, and documentation discipline. The end-to-end flow is presentation-friendly: owner upload, admin validation, executor read-only release simulation, and automated reporting. Scope is intentionally bounded to ensure delivery by 29 May while preserving extensibility for future automation, workflow integration, and enhancements.
