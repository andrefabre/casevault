# ICT171 MVP Clean Base

This branch is a minimal base for the ICT171 Digital Legacy Vault build.

Current behavior:
1. Next.js root route (`/`) redirects to `/index.html`.
2. The starter UI is served from `public/index.html` with `public/css/style.css` and `public/js/main.js`.

## Quick Start

```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Kept on Purpose

1. `app/layout.tsx` and `app/page.tsx` for Next routing.
2. `app/globals.css` minimal reset.
3. `public/` static starter files.
4. Core Next/React/TypeScript config files.
